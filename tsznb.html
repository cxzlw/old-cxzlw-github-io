远程加密命令执行

命令执行器：Eval

Python 中有这么一种代码：

eval("print('hello world')")

eval() 函数用来执行一个字符串表达式，并返回表达式的值。

菜鸟教程

但是这行代码会报错：

eval("import requests")

原因是 eval() 函数仅支持调用函数，不支持部分关键词调用

eval的妙用

上文提到 eval() 仅支持函数的调用，但是要调用关键词我们可以这么写

eval("exec('import requests')")

为了方便演示我们暂时只是用简单的字符移位操作😀

加密函数（简易版）

def encode_easy(raw: str) -> str:
    return "".join([chr(ord(x) + len(raw) // 2 + 1) for x in raw])

解密函数（简易版）

def decode_easy(ecd: str) -> str:
    return "".join([chr(ord(x) - len(ecd) // 2 - 1) for x in ecd])
